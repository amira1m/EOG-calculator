EOG-Based Calculator
A Python application that uses Electrooculography (EOG) signals — eye movement tracking — to control a calculator without touch or keyboard input.
What it does
Detects eye movement signals (EOG) and maps them to calculator operations, allowing a user to navigate and operate a calculator purely through eye movements. This is an accessibility-focused project exploring hands-free human-computer interaction.
How it works

Signal Input — Reads EOG signals that correspond to eye movement directions (left, right, up, down, blink)
Signal Processing — Processes and interprets the raw signals to detect intentional eye movements
Calculator Control — Maps the detected eye movements to calculator navigation and input actions

Use Case
Designed as an assistive technology concept for users with limited motor abilities, demonstrating how bio-signals can be used to control software interfaces.
